MC203 Project Instructions

Follow the instructions below to complete the project:

  1. Download the project files from the course website.
  2. Extract the files to a folder on your computer.
  3. Open the index.html file in a web browser to view the project.
  4. Use a text editor to modify the HTML and CSS files as needed.
  5. Test the project in a web browser to ensure it works correctly.
  6. Submit the project files to the course website for grading.
